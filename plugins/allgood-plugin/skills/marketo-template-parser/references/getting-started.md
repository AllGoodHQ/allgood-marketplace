# Getting Started

## What You Need

- A Marketo Email 2.0 template file (e.g., `my-template.html`)
- Python 3 with beautifulsoup4: `pip3 install beautifulsoup4`

## Using with Claude Desktop

For step-by-step Claude Desktop setup instructions, see:
https://www.allgoodhq.com/blog/how-to-use-claude-desktop-for-data-analysis

## What Can I Do?

I can help you with:

1. **Generate reference docs** - Create overview and detailed documentation
2. **List all modules** - See what modules are available
3. **Get module details** - Deep dive into specific modules
4. **List variables** - Show all customization variables
5. **Validate template** - Check for errors and best practices
6. **Answer questions** - Learn about Marketo syntax and standards

Just tell me what you'd like to do! For example:
- "What modules are available?"
- "Generate a reference"
- "Validate the template"
- "What's in the hero1 module?"
